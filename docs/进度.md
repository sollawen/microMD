# micromd 项目进度

## 已完成

### 阶段1：Markdown 表格对齐渲染

为 micro 编辑器添加了只读的表格列对齐显示功能。Buffer 数据不变，仅在渲染时通过注入 padding 让 `|` 对齐。

#### 新增文件

| 文件 | 说明 |
|------|------|
| `internal/display/markdown_table.go` | 表格检测、列宽计算、padding 逻辑（含 `ProcessRune` 方法） |
| `internal/display/markdown_table_test.go` | 单元测试（39 个用例） |

#### 修改文件

| 文件 | 改动量 | 说明 |
|------|--------|------|
| `internal/display/bufwindow.go` | +46/-2 行 | 调用 `DetectTables` + `ProcessRune` + 绘制 padding |
| `internal/config/settings.go` | +1 行 | 添加 `mdtablealign` 配置项 |

#### 功能

- 自动检测 Markdown 表格（需有分隔行 `|---|`）
- 各列按最大宽度对齐，支持中英文混排（CJK 字符按 2 宽度计算）
- 分隔行用 `-` 填充对齐，内容行用空格填充
- 正确处理 `\|` 转义竖线
- 跳过 fenced code block（``` / ~~~）内的表格
- 继承 cursor-line 背景色，视觉不断裂
- 表格行禁用软换行（Phase 1 简化处理）

#### 已知限制

- 光标位置不对（屏幕上多了空格，光标仍按 buffer 坐标）
- 水平滚动时回退原始渲染
- 每帧重新检测表格（无缓存）
